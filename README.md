# forecast.riktom.com

Research-weighted 24-hour hunt + fish activity forecast.

Two scoring profiles over the same data inputs (weather, solunar, moon phase, fronts):

- 🎣 **Fish** — calibrated for warm-water species (largemouth, catfish, crappie, bream)
- 🦌 **Hunt** — whitetail-tuned (dawn/dusk movement, scent-control wind, front passage)

Built on Open-Meteo (free, no API key). Fish profile is a port of [`Tnijem/hahirafish`](https://github.com/Tnijem/hahirafish).

See [`CLAUDE.md`](CLAUDE.md) for architecture, algorithm details, and deployment.
